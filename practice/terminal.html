Introduction

Today, we will be learning about how to use the terminal. the terminal is a very useful tool built right into every computer. However, if you dont know how to use it, then it is just a black screen with some random words and symbols on it.


How to use the Terminal

In order to use the terminal, you need to know some commands.

  • cd is used to move to a new folder. For example, if I am in at ~ (home directory) and type cd Desktop/, I will now be in the Desktop folder.
  • ls is used to look around. If I am still in the Desktop folder from when we used cd, and type ls, I can see everything on my desktop.
  • Now that we are in Desktop and can see what is in it, we can start deleting stuff. To delete something, do rm filename. Poof. Now its gone.
  • To make a new directory (the terminal calls folders directories) we do mkdir directoryname. Now you have a new folder called directoryname.

These are just some of the basic commands, but they should allow you to move around, and not feel totaly lost.


Vim

Vim is how we edit files. It is a seperate branch of the terminal, and is the single most usefull tool you can have. unfortumately, it can get pretty annoying if you dont know what you are doing. Here are the steps you will have to do in order to make your first file using vim.

  1. To create a file named thingamabob.html, type vim thingamabob.htm. You should now be in a weird black screen with a bunch of brakets. This is vim.
  2. To actually write something in vim, you need to press i or o. Now you can write whatever you want.
  3. To leave vim, press esc and type :wq. The : tells vim that you are about to write a command, and w saves what you wrote, and q quits vim.

Conclusion

This concludes a very basic, very brief introduction to the terminal and vim. If you want to learn more about the terminal, there is a game called terminus that explains it very well. And if you want to learn more about vim, go into the terminal and write vimtutor. this opens a vim tutorial that takes around 30 minutes to finish.
